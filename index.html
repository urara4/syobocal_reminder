Syobocal reminder

しょぼいカレンダーの番組情報をつぶやくtwitterクライアント

Download this project as a .zip file Download this project as a tar.gz file

つぶやき内容

  • 前日22:00頃に、翌日4:00～28:00に放送される初回放送(再放送含む)・特番の番組情報をツイートします。
  • 番組開始5分前に番組情報をツイートします。
番組情報は以下のサイトから取得しています(30分1回チェック)。
しょぼいカレンダー
ハッシュタグは以下のサイトから取得しています(1日1回チェック)。
ハッシュタグ一覧まとめ - zish - テレビ実況に特化したTwitterハッシュタグまとめデータベース

諸注意

2014/01/23時点では多数バグがあると思われますので、安定するまでしばらくお時間を下さい。
現状リフォロー機能がありません。実装不要かと考えていますが、必要な方はリプをお願いします。
現状長タイトル作品で投稿漏れを起こす可能性があります。
所詮botなので、Twitter社のアクションでアカウント削除・凍結があり得ます。
しょぼカレの情報が誤っている場合でも、特段修正は加えていません。
